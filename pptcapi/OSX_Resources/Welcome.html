About the OPeNDAP 4 Data Server (Hyrax) BES Package

This package provides the a C API for the BES PPT protocol to allow users to develop C appliations to talk to the OPeNDAP BES server. This includes the libraries and header files required to build against.

To install BES PPT C API, you'll need to install and configure this software

See the OPeNDAP Documentation Wiki for more information.
