About the OPeNDAP 4 Data Server (Hyrax) BES Package

This package provides the basic functionality for the Hyrax data server BES from OPeNDAP. This includes the libraries required to run the BES as well as those needed to develop new components for use with the server. The BES (Back End Server) is one of two separate processes that make up Hyrax.

To install Hyrax, you'll need to install and configure this software and the front end to the server as well.

See Hyrax for more information.
